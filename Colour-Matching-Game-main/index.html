How To Play

1. Once you begin the game you will see a set of black sqaures

2. You will need to select any black sqaure of your choice and it will show you a color

3. You will need to find where the other sqaure with the same colour is on the board by selecting a second sqaure

4. Once you get a match the sqaures you picked will stay open as a matched set

5. Match all the sqaures to each other to move on to the next level
